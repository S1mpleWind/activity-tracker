# Activity Tracker

一个简单易用的 PC 活动追踪工具，帮你了解自己的电脑使用习惯。

## 它能做什么？

- **自动记录**：后台运行，记录你使用的每个应用程序
- **使用统计**：查看每天在不同应用上花了多少时间  
- **可视化报告**：用图表直观展示你的使用情况
- **数据持久化**：所有数据都保存在本地数据库中

## 适合谁用？

- 想了解自己时间花在哪里的用户
- 需要统计工作时间的自由职业者
- 关心数字健康，希望减少无效使用的人
- 对个人数据分析感兴趣的爱好者

## 主要特点

- 完全离线运行，保护隐私
- 轻量级，不影响系统性能
- 简单直观，开箱即用
- 支持 Windows 系统( macOS , Linux 版开发中)

只需运行程序，它就会在后台默默记录，随时可以查看详细的使用报告。
