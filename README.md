# Activity Tracker

一个简单易用的 PC 活动追踪工具，帮你了解自己的电脑使用习惯。

所有活动的数据都存储在你的本地的数据库里，不用担心隐私泄露

## 使用指南

### （1）.直接下载release并解压，点击gui.exe运行

### （2）.使用源码运行

#### 0.拥有一个python并将代码拉取到你的本地

#### 1.(创建一个conda环境)， 并`pip install requirementlist`

#### 2.运行`python gui.py`
有关GUI的使用介绍：

0. 这是一个简陋的gui

1. *dashboard*界面会可以开始/关闭追踪，并删除你选取的一段时间的数据库记录

2. *analysis*界面
   1. 可以使用`tpday`来获取今天的数据
   2. 可以输入起始的时间，使用`range`来获取这一段时间的数据
   3. 使用`refresh data`来更新当前的显示


## 它能做什么？

- **自动记录**：后台运行（正在开发中），记录你使用的每个应用程序
- **使用统计**：查看每天在不同应用上花了多少时间  
- **可视化报告**：用图表直观展示你的使用情况
- **数据持久化**：所有数据都保存在本地数据库中

## 适合谁用？

- 想了解自己时间花在哪里的用户
- 需要统计工作时间的自由职业者
- 关心数字健康，希望减少无效使用的人
- 对个人数据分析感兴趣的爱好者

## 主要特点

- 完全离线运行，保护隐私
- 轻量级，不影响系统性能
- 简单直观，开箱即用
- 支持 Windows 系统( macOS , Linux 版开发中)
